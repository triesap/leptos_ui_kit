# leptos_ui_kit

Source-first UI kit for Leptos with installable component source and a CLI.

## Goals

- Provide a consistent, Leptos-native component and theming foundation.
- Install components as project-owned source files.
- Use a CLI to add, inspect, and update components.
- Keep primitives, styling, and accessibility behavior predictable.
